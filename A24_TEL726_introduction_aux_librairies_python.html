2. √âgalisation d‚Äôhistogramme

L‚Äô√©galisation d‚Äôhistogramme est une technique importante en traitement d‚Äôimage pour am√©liorer le contraste global d‚Äôune image. Voici les principes cl√©s de cette m√©thode :

Principe de base

L‚Äô√©galisation d‚Äôhistogramme vise √† redistribuer les intensit√©s des pixels d‚Äôune image de mani√®re plus uniforme sur toute la plage de valeurs disponibles[1][6]. L‚Äôobjectif est d‚Äôobtenir un histogramme plus √©quilibr√©, ce qui se traduit g√©n√©ralement par une am√©lioration du contraste de l‚Äôimage.

√âtapes du processus

  1. Calcul de l‚Äôhistogramme initial : On d√©termine la distribution des intensit√©s de pixels dans l‚Äôimage d‚Äôorigine.

  2. Calcul de l‚Äôhistogramme cumul√© : On calcule l‚Äôhistogramme cumul√©, qui comptabilise pour chaque niveau de gris le nombre total de pixels ayant une intensit√© inf√©rieure ou √©gale[4].

  3. Transformation des intensit√©s : On applique une fonction de transformation non lin√©aire bas√©e sur l‚Äôhistogramme cumul√© pour modifier les intensit√©s des pixels[1].

  4. Redistribution des intensit√©s : Les nouvelles intensit√©s sont r√©parties sur toute la plage disponible, g√©n√©ralement de 0 √† 255 pour une image en niveaux de gris 8 bits[1].

Effets sur l‚Äôimage

  • Am√©lioration du contraste : L‚Äô√©galisation d‚Äôhistogramme augmente le contraste global de l‚Äôimage, en particulier pour les images √† faible contraste initial[8].

  • √âtalement de la dynamique : Les valeurs d‚Äôintensit√© sont √©tal√©es sur toute la plage disponible, ce qui peut faire ressortir des d√©tails auparavant peu visibles[3].

  • Uniformisation de la distribution : L‚Äôhistogramme r√©sultant tend vers une distribution plus uniforme, bien que parfaitement plate en pratique[2].

Consid√©rations importantes

  • Cette technique est particuli√®rement efficace pour les images sombres ou √† faible contraste.
  • Elle peut parfois produire des r√©sultats peu naturels ou exag√©rer le bruit dans certaines images.
  • Pour les images couleur, l‚Äô√©galisation est g√©n√©ralement appliqu√©e sur la composante de luminance uniquement.

En r√©sum√©, l‚Äô√©galisation d‚Äôhistogramme est une m√©thode puissante pour am√©liorer automatiquement le contraste d‚Äôune image en redistribuant ses intensit√©s de mani√®re plus uniforme, ce qui peut r√©v√©ler des d√©tails auparavant difficiles √† percevoir.

def image_histogram_equalisation(image):
    # get image histogram
    image_histogram, bins = np.histogram(image.flatten(), 256, normed=True)
    cdf = image_histogram.cumsum() # cumulative distribution function
    cdf = (255-1) * cdf / cdf[-1] # normalize
    # use linear interpolation of cdf to find new pixel values
    image_equalized = np.interp(image.flatten(), bins[:-1], cdf)

    return image_equalized.reshape(image.shape).astype('uint8')
img_he = image_histogram_equalisation(img[:,:,::-1])

import numpy as np
import rasterio
from scipy.stats import norm

def gaussian_cdf(size=256):
    x = np.linspace(-3, 3, size)
    y = norm.cdf(x)
    return y

def cdf(im):
    """
    Computes the CDF of an image
    """
    hist, _ = np.histogram(im.flatten(), 256, [0, 256])
    cdf = hist.cumsum()
    return cdf / cdf[-1]

def hist_matching(c, c_t, im):
    """
    c: CDF of input image
    c_t: CDF of template (Gaussian in this case)
    im: input image
    """
    pixels = np.arange(256)
    new_pixels = np.interp(c, c_t, pixels)
    return new_pixels[im].astype(np.uint8)
